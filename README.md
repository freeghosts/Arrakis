# Arrakis

Arrakis is a console RPG written in Java

### Works On

Linux, Mac, Windows 10.19+

>[!NOTE]
>You will need the Jansi color library for Arrakis to work on older versions of Windows: [Download Jansi](https://fusesource.github.io/jansi/)

### Run On Terminal

```java
javac game.java
java game
```
